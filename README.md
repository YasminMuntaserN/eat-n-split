# 🧑‍🤝‍🧑Eat n Split
## 📘 About The Project
React app to split bills, track shared expenses in a group of friends easily and find out "Who owes whom and how much?" with a transaction minimisation algorithm.

## 🛠️ Built In
- **React**
- **CSS**
- **Javascript**
